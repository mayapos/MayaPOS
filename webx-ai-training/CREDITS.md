## Technology Stack

### Harbour Project
**Website:** https://harbour-project.org
**License:** GPL v3.0.0 with linking exception
**Contribution:** Core Harbour compiler and runtime used by WebX

The WebX framework is built on Harbour, an open-source xBase compiler. While WebX itself is proprietary, it uses Harbour's GPL-licensed compiler with the linking exception that allows commercial applications.

### FiveWin Compatibility
**Note:** WebX provides FiveWin-compatible API signatures but contains **NO FiveWin source code**. All implementations are original work by Byron Henderson/MayaPOS.

**Compatibility approach:** API-compatible method signatures only (legal under Oracle v. Google precedent)

## Acknowledgment of Standing on Shoulders

This work builds upon decades of innovation:

- **Clipper** (1985-1992) - Pioneer of xBase compiled applications
- **dBase** (1979+) - Original xBase database language
- **FoxPro/Visual FoxPro** (1984-2015) - Microsoft's xBase evolution
- **Harbour** (1999+) - Modern open-source xBase compiler
- **FiveWin** (1993+) - Windows GUI framework for xBase
- **mod_harbour** (2017+) - Web deployment for Harbour

WebX stands on these shoulders, bringing xBase to the modern web with AI-native development.
